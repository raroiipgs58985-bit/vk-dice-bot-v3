# Кубятня — модуль экономики v4.0

Скопируйте файлы из архива в корень репозитория с сохранением папок:

- `bot.py` — заменить существующий;
- `config.py` — заменить существующий;
- `modules/__init__.py` — новый файл;
- `modules/economy.py` — новый файл.

## Переменные Render

Обязательно:

```text
VK_TOKEN=токен_группы
VK_GROUP_ID=идентификатор_группы
VK_ADMIN_IDS=ваш_vk_id
```

Несколько администраторов:

```text
VK_ADMIN_IDS=123456789,987654321
```

Необязательно:

```text
STARTING_BALANCE=100
SALARY_MIN=10
SALARY_MAX=20
SALARY_COOLDOWN_SECONDS=86400
```

## Команды администраторов

```text
[событие Оборона улья | Победа | Поражение
[закрыть ставки
[итог Победа
[отмена события
[выдать [id123456|Игрок] 50
[забрать [id123456|Игрок] 25
[установить [id123456|Игрок] 100
[сброс ставок
[сброс тронов
[экономика
```

Команды выдачи, изъятия и установки баланса также работают ответом на сообщение игрока.

## Важно

Данные хранятся только в оперативной памяти процесса. При перезапуске или новом деплое Render:

- балансы обнуляются;
- активное событие исчезает;
- новые игроки снова получают стартовый баланс.
