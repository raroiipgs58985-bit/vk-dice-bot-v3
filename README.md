# Кубятня 3.0

Финальная модульная версия VK-бота.

## Возможности

- броски кубиков: `[к100`, `[3к20+5`, `[3к20-5`;
- несколько бросков отдельными строками;
- комментарии через `#`;
- одна неповторяющаяся цитата на весь отчёт;
- проверки к100 по порогу: `[1к100 (50) #стрельба`;
- подсчёт степеней успеха и провала;
- 200 описаний талантов;
- поиск талантов: `[таланты мгновенная`;
- текстовые и графические триггеры без КД;
- кэширование загруженных во ВКонтакте изображений;
- `/health` для Render.

## Установка поверх существующего репозитория

1. Распакуй архив.
2. Скопируй все файлы в корень репозитория с заменой.
3. Сохрани существующие PNG-картинки в корне репозитория.
4. Установи зависимости:

```text
pip install -r requirements.txt
```

5. Команда запуска Render:

```text
python bot.py
```

## Картинки

PNG-файлы из старого репозитория в архив не включены. Они должны остаться рядом с `bot.py`.

Список текущих имён указан в `data/replies.json`.

## Добавление триггера

Открой `data/replies.json`:

```json
"император": {
  "text": "Император защищает!",
  "image": "Emperor.png",
  "attachment": null
}
```

Картинку `Emperor.png` положи рядом с `bot.py`.

После изменения перезапусти или передеплой бота.

## Добавление таланта

Открой `data/talents.json`:

```json
"Название таланта": "Описание таланта"
```

Требования в базе не используются.

## Команды талантов

Точное описание:

```text
[мгновенная реакция
```

Поиск по части названия:

```text
[таланты мгновенная
```

## Проверка к100

```text
[1к100 (50) #стрельба
```

При пороге 50:

- 24 — 3 успеха;
- 44 — 1 успех;
- 50 — 1 успех;
- 51 — 1 провал;
- 60 — 2 провала;
- 79 — 3 провала.

## Настройки Render

Рекомендуется задать переменные:

```text
VK_TOKEN
VK_GROUP_ID
```

Для совместимости текущие значения оставлены запасными в `config.py`.
