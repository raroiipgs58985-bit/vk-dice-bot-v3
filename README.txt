Кубятня 4.0.1 — настройки напрямую из config.py

1. Замените текущий config.py этим файлом.
2. Откройте config.py.
3. В TOKEN вставьте новый токен сообщества VK.
4. В VK_ADMIN_IDS впишите числовые VK ID администраторов.
5. Сохраните изменения и выполните Deploy/Restart на Render.

Переменные VK_TOKEN, VK_GROUP_ID и VK_ADMIN_IDS в Render больше не нужны.

Важно:
старый токен был опубликован в репозитории. Его лучше отозвать в настройках сообщества VK
и создать новый, иначе им может воспользоваться посторонний.
