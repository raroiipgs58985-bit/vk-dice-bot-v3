# Подключение отдельного агента к «Кубятне»

Этот комплект не переносит сканер в основной бот. Он добавляет только HTTP-клиент, который по команде пользователя будит отдельный Render и затем публикует его ответ в тот же диалог ВК.

## Заменить в текущем репозитории

- `bot.py`
- `requirements.txt`
- `modules/__init__.py`

## Добавить

- `modules/site_agent_client.py`

`config.py`, токен ВК, экономика, цитаты, таланты и папка `data` не изменяются.

## Переменные Render основного бота

```text
SITE_AGENT_URL=https://vk-site-ai-agent.onrender.com
SITE_AGENT_SECRET=тот-же-секрет-что-у-агента
```

Необязательные:

```text
SITE_AGENT_CONNECT_TIMEOUT=90
SITE_AGENT_READ_TIMEOUT=600
SITE_AGENT_PARALLEL_REQUESTS=2
```

## Команды ВК

```text
[найди вопрос
[агент вопрос
[глубокий поиск вопрос
[агент статус
```

Поиск выполняется в отдельном потоке, поэтому броски, ставки и прочие команды не блокируются.
